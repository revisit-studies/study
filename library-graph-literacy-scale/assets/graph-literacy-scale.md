
# graph-literacys-scale



This is an example study of the library `graph-literacy-scale`.

The Graph Literacy Scale is a 13-item instrument that measures peopleâ€™s ability to understand common graphical formats, with a focus on health-related graphs. This library contains 13 components, each corresponding to a question of the Graph Literacy Scale. It also includes a sequence of all 13 components (the full Graph Literacy Scale).
